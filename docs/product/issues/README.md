# Product Issues

This directory breaks the roadmap into milestone-ready planning files that can be turned into tickets or tracked as delivery epics.

## Milestones
- `milestone-1-playable-solo-mvp.md`
- `milestone-2-hosted-multiplayer-core.md`
- `milestone-3-content-operations-and-persistence.md`
- `milestone-4-team-play-and-competitive-extensions.md`

Each file includes:
- the user outcome
- scope boundaries
- ranked work items
- dependencies
- risks
- acceptance criteria
